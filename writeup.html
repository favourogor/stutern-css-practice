What I Have Learnt so Far

Since the beginning of my journey to becoming a Frontend web developer, I have learnt quite a lot. I have learnt that Frontend web development is an aspect of web development that deals with the client side of a web page. In other words, it deals with the frontend aspect of a web page or web a web application, that is, how the layout of a web page or web app looks. A Frontend web developer works closely with a UI/UX designer and a Backend web developer. In bringing a web page to live, HTML (HyperText Markup Language), CSS (Cascading Style Sheet) and Javascript are used. HTML gives the page a structure. CSS is used in styling the page structure. It makes the frontend of a website shine and it creates a good user experience. Without CSS, websites and web apps would be less pleasing to the eye and most likely, much harder to navigate. CSS is responsible for layout, format, color, and so on.

Margin, Border and Padding (Differences)

  • Padding:The inner space beetween the content and Border of your box.
  • Border:The perimeter of the box. Borders can be invisible or they could be a thin colored line.
  • Margin:The outer space (or lack of space) surrounding the box.
